# TypeScript with Interfaces to JS dist using Lodash

  Install TypeScript:

```bash
$ npm i -g typescript
```
  Install [Http-server](https://www.npmjs.com/package/http-server)

```bash
$ npm i -g http-server
```

  Compile to JS:

```bash
$ tsc
```

  Run Server:

```bash
$ http-server -c-1
```

__NOTE__: you can install Lodash with:

```bash
$ npm install --save @types/lodash
```
